Description détaillée

La classe de configuration de débogage permet d'afficher divers informations en lien avec le jeu :

  • apparition d'un astéroïde
  • mode d'un joueur virtuel, s'il y a lieu
  • type d'éclairage

Des effets visuels peuvent également être activés depuis cette classe.

Toutes les sorties sont horodatées.
