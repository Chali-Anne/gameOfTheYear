Description détaillée

Gère l'affichage de texte basé sur une image de caractères générés par Bitmap Font Generator v1.12 (http://www.angelcode.com/products/bmfont/). L'image doit être de type Windows bitmap 8-bit, et les pixels "allumés" doivent être de couleur à l'index 0x00 de la palette du fichier. Le fichier de description ".fnt" doit être de type texte. Ne gère pas le kerning inter-caractère pour l'instant.

Auteur
Philippe Proulx
